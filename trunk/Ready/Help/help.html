Help Menu

The Help menu items can be used to display information on a variety of topics. Each item corresponds to a .html file in the Help folder. The information is displayed in the Help Pane which behaves like a simplified browser.

Keyboard shortcuts

Various keyboard shortcuts can be used when the Help Pane has the focus:

  • Hit "+" and "-" to change the font size.
  • Hit the left arrow or "[" to go back.
  • Hit the right arrow or "]" to go forwards.
  • Hit the home key to go to the Contents page.
  • Hit the up/down arrow keys or page up/down keys if a scroll bar appears.

Special links

The Help Pane supports a number of Ready-specific links:

  • edit:file
    Open the given file in your preferred text editor. A non-absolute path is relative to the location of the Ready application.
  • open:file
    Load the given pattern. A non-absolute path is relative to the location of the Ready application. Ready will open the file in your text editor by control-clicking or right-clicking on the link.
  • prefs:pane
    Open the Preferences dialog at the given pane. The pane string must be lowercase and must match a known pane: file, edit, view, etc.
