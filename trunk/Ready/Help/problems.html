Known Problems

This page lists all known bugs and limitations. If you come across any problems not mentioned below then please report them to reaction-diffusion@googlegroups.com.

Problems on all platforms

  • use_image_interpolation doesn't work on images when show_cell_edges is on.
  • show_cell_edges doesn't work on the 2D slice through 3D images (slice_3D).

Mac problems

  • Toolbar tips can't be disabled (due to bug in wxOSX-Cocoa).

Windows problems

  • Terminal windows appear when loading an OpenCL pattern, on some platforms. (OpenCL driver issue?)

Linux problems

  • The color scale sometimes appears with garbled banded colors on Debian. (Graphics driver issue?)
  • Can't tab between edit controls in XYZ dialogs on Xubuntu.
