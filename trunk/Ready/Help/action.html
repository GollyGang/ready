Action Menu

Step by 1

Advances the pattern by a single timestep.

Step by N

Advances the pattern by the current number of timesteps per render. This number can be set via Change Running Speed (see below) or by editing the render setting in the Info Pane.

Run/Stop

Starts or stops simulating the current pattern.

Reset

Restores the starting pattern.

Generate Initial Pattern

Similar to New Pattern but creates an initial pattern.

Run Faster

Doubles the number of timesteps taken between each render.

Run Slower

Halves the number of timesteps taken between each render.

Change Running Speed...

Allows the number of timesteps taken between each render to be specified manually.

Add Parameter...

Adds a new parameter, first asking for the name.

Delete Parameter...

Deletes a parameter, first asking for the parameter to delete.

Select OpenCL Device...

Opens a dialog so you can select which OpenCL device will be used.

Show OpenCL Diagnostics...

Displays information about all the available OpenCL devices.
