Detailed Description

An RD system that uses an OpenCL formula snippet.

An N-dimensional (1D,2D,3D) OpenCL RD implementations with n chemicals specified as a short formula involving delta_a, laplacian_a, etc. implemented with Euler integration, a basic finite difference stencil and float4 blocks for speed
