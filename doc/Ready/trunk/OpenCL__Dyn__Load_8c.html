Function Documentation

cl_int CL_API_CALL clLibLoad ( )

Loads the OpenCL library if available

This function must be called prior to calling anu other cl*() function

Returns:
Returns CL_DEVICE_NOT_AVAILABLE if the OpenCL Library is not found
CL_DEVICE_NOT_AVAILABLE if a library function is not found.
and CL_SUCCESS if all modules are loaded succesfully.
