Known Problems

This page lists all known bugs and limitations. If you come across any problems not mentioned below then please add then to https://github.com/GollyGang/ready/issues or send a mail to reaction-diffusion@googlegroups.com.

Problems on all platforms

  • show_cell_edges shows triangles on the 2D slice through 3D images (slice_3D).

Mac problems

  • Toolbar tips can't be disabled (due to bug in wxOSX-Cocoa).

Windows problems

  • Panning behaves strangely on complex scenes, e.g. Bays_3D.vti.

Linux problems

  • Can't tab between edit controls in XYZ dialogs on Xubuntu.
